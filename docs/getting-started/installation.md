# Installation

Install the core SDK and the React bindings using your preferred package manager.

=== "npm"

    ```bash
    npm install @axsdk/core @axsdk/react
    ```

=== "pnpm"

    ```bash
    pnpm add @axsdk/core @axsdk/react
    ```

=== "yarn"

    ```bash
    yarn add @axsdk/core @axsdk/react
    ```

=== "bun"

    ```bash
    bun add @axsdk/core @axsdk/react
    ```
